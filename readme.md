# schnittstabil/phpunit-starter

> Starts phpunit with my opinionated phpunit.xml.dist.


## Install

```sh
$ composer require --dev schnittstabil/phpunit-starter
```


## Usage

```json
{
    ...
    "require-dev": {
        "schnittstabil/phpunit-starter": "..."
    },
    "test": [
        "schnittstabil-phpunit-starter"
    ]
}
```
